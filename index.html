SI Pengelolaan Barang

Project ketika Magang di kantor Kementrian Agama Kabupaten Trenggalek. Sistem Informasi Pengelolaan Barang ini dibuat berdasarkan kebutuhan dan permintaan pihak operator gudang yang sebelumnya masih manual menggunakan buku.

Fokus utama sistem ini adalah untuk pencatatan barang masuk dan keluar serta memonitoring stok barang secara otomatis baik untuk menambahkan data masuk/keluar, maupun ketika mengubah data ketika ada kesalahan, serta laporan secara otomatis setiap bulannya yang di export sebagai file Excel.

Sistem ini dibuat dengan menggunakan PHP(Laravel 7), Mysql, Bootsrap, Serta Ajax untuk CRUD secara Realtime.
